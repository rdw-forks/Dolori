Dolori
=======

External dependencies
---------
* OpenGL
* DevIL
* SDL2_ttf

Build
-----
```shell
$ git clone https://gitlab.com/Linkz/Dolori
$ cd Dolori
$ mkdir build && cd build
$ cmake ..
$ cmake --build . --config MinSizeRel
```

Current supported settings
--------------------------
* Packet version: 20080910
* Packet obfuscation: Disabled
* Client version: 23
